A step-by-step guide to building a hardware tool inventory app with Mendix

1. Introduction

This guide provides you with a step-by-step walkthrough to build a hardware tool inventory application using Mendix Studio Pro. You'll learn how to create entities, generate pages, configure navigation, and test your application.

2. Prerequisites

Before getting started, complete the following prerequisites:

  1. Login to your Mendix account created using the credentials provided.
  2. Download and Install Mendix Studio Pro 10.24.x (LTS).
  3. Install and configure Parallels or install the 10.24.x beta version for MacOS.

4. Extending the App

4.1 Building the Homepage

To build and extend the app, perform the following steps:

  1. Select and right-click to delete all the contents on the Home_Web page.
  2. In the right dockable pane, click Toolbox and click Widgets.
  3. From the Structure dropdown, drag and drop Containers to the homepage.
  4. From the Text dropdown, drag and drop the Page Title. Click the Topbar and rename it to Chadwick's Hardware Inventory.
  5. Open the Cards dropdown, then drag and drop a Card Action block onto the homepage. Rename the card to Tools.
  6. Click Properties and navigate to Styling. Adjust the card spacing to your preference.
  7. Click the button icon and configure the following properties:
    • Icon: add-circle
    • Render mode: Button
    • Button style: Primary
  8. Style the button with these settings:
    • Spacing: Margin Left, Right: Left
    • Size: Large
    • Border: Enabled
  9. To preview the homepage, click Run Locally and click View App.

4.2 Building the Tools Page

Creating the Entity

  1. In the App Explorer, expand the MyFirstModule dropdown and click Domain Model.
  2. Drag a new Entity from the Toolbox into the domain model. Alternatively, right-click anywhere in the Domain Model and select Add entity. You have created a persistable entity; all tool information will be stored in a database.
  3. Double-click the entity to open the properties. Name the entity Tool.
  4. To create a new attribute, click New. This will open a dialog box.
  5. Create four attributes as shown in the table below:
Name Type
Name String
Code String
Quantity Integer
Price Decimal
There are four types of entities. Learn more about entities here.
  1. Click OK.

You have now created an entity.

Generating Overview Pages

To create the tools overview and details page, perform the following:

Note: There are multiple ways to generate overview pages. For more information refer to Creating Overview Pages.
  1. Right-click the Tool entity and click Generate overview pages, then click OK.
  2. Click the OverviewPages folder to view the pages.
  3. Click the Tool_Overview page and click the New Tool button to adjust the spacing in the Properties.
  4. Double-click the New Tool button to open the edit action button properties and rename the Caption to Add a new tool.
  5. Double-click the Tool text and rename it to Tools Overview.
  6. Select all the text filters and click Delete.
  7. In the App Explorer, click Tool_NewEdit.
  8. Click each attribute to open the properties dialog box and set the Validation Type to Required.

Configuring Navigation

You have now created the three pages required. To create navigation between the pages, perform the following steps:

  1. Click Navigation in App Explorer.
  2. Click New Item from the Menu section.
  3. Set the Caption to Navigation and set the Icon to compass-directions, then click Select.
  4. Set On click to Show a page. From the Select a web page dialog box, click Home_Web and click Select.

In the Home Page, perform the following steps to configure the card action button to open the OverviewPages:

  1. Double-click on the button to open the Edit Action Button dialog box. Select Show a page in the On Click option under the Events section.
  2. Expand OverviewPage and select Tool_Overview, then click Select.
âœ… You have successfully configured the button to open the Tools page.

5. Testing the App

To test the app, follow the instructions given below:

  1. Click the Run Locally icon on the Run Menu.
  2. Once the app is running, click View App. The app is deployed locally on your machine.

6. Adding Sample Data

To add sample data to your app, perform the following steps:

  1. Click the Add button.
  2. Click Add New Tool.
  3. Enter the Name, Code, Quantity, and Price of the tool and click Save.
